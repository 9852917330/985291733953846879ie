# IELTS 9.0 PWA — v46

Bản v46 làm lại toàn bộ dàn ý theo chuẩn ngắn gọn, súc tích và đúng mạch triển khai:

- Không cắt câu từ bài mẫu.
- Không nhặt keyword rác.
- Không copy nguyên câu tiếng Anh vào dàn ý.
- Dàn ý viết bằng tiếng Việt, nhìn qua là biết nói/viết theo hướng nào.
- Speaking Part 2: đối tượng → mô tả/bối cảnh → chi tiết chính → lý do/cảm xúc.
- Speaking Part 1/3: từng câu hỏi có ý trả lời lõi + hướng mở rộng.
- Writing Task 2: mở bài → luận điểm → cơ chế/ví dụ → kết luận.
- Writing Academic: mở bài → overview → nhóm dữ liệu/quy trình/bản đồ.
- Writing General: mở thư → từng bullet đề bài → kết thư.
- Service Worker cache: v46.

Upload đè toàn bộ file trong ZIP lên GitHub Pages, sau đó mở `index.html?v=46` hoặc gỡ/cài lại app nếu điện thoại còn cache cũ.
